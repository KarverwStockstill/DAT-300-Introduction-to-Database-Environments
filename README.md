# DAT-300-Introduction-to-Database-Environments
Introduction to Database Environments 

In this course, I learned about data quality management, the ETL process (Extract, Transform, Load), and database concepts. Developing skills in data validation and cleanup, including identifying missing data, handling scientific notation, and standardizing inconsistent formats across data sets. I also learned how to analyze sales and inventory data, such as identifying sales trends and peak revenue months to support business decisions. In addition, I gained experience using business intelligence tools like Microsoft Power BI to track metrics and communicate insights clearly to stakeholders. I used the gap analysis business method to compare an organizations current performance with its desired goals.
